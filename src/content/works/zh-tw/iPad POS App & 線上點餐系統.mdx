## iPad POS App

- 基本交易紀錄、點餐功能
- 菜單編輯功能
- 線上點餐新訂單通知功能
- 使用 Google 帳號登入

## 線上點餐系統

- 線上點餐
- 使用 Google 帳號登入自動帶入會員資訊

## 技術亮點

- 架設 MinIO 作為 Object Storage 儲存圖片檔案
- 使用 Docker Swarm 搭配 Cloudflare CDN 部署線上點餐系統
- 使用 Firebase Firestore 做為資料庫，即時偵測資料變化，跨裝置即時同步
